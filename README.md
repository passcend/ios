# Passcend iOS

---

Passcend iOS 비밀번호 관리자 및 인증 앱입니다.

## 시작하기

빌드 지침, 권장 도구, 코드 스타일 팁 등에 대한 자세한 내용은 프로젝트 문서를 참조하세요.

### 요구사항

- Xcode 15+
- iOS 15.0+
- Swift 5.9+

## 관련 프로젝트

- **server/**: 핵심 인프라 백엔드 (API, 데이터베이스, Docker 등)
- **clients/**: 비모바일 클라이언트 애플리케이션
- **directory-connector/**: 디렉토리 동기화 도구 (AD, LDAP, Azure, G Suite, Okta)

## 기여하기

코드 기여를 환영합니다! `main` 브랜치에 대해 Pull Request를 제출해주세요.

보안 감사 및 피드백을 환영합니다. 민감한 보안 문제는 비공개로 보고해주세요. 보안 정책은 [`SECURITY.md`](SECURITY.md) 파일을 참조하세요.
